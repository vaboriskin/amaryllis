```
│   │   │   └── Home/

│   │   ├── utils/         # Утилиты
│   │   ├── types/         # TypeScript типы
│   │   ├── App.tsx
│   │   └── main.tsx
│   ├── public/            # Статические файлы
│   └── package.json
│
├── backend/               # Backend API (Express + TypeScript)
│   ├── src/
│   │   ├── routes/        # API маршруты
│   │   ├── controllers/   # Контроллеры
│   │   ├── models/        # Модели данных
│   │   ├── middleware/    # Middleware
│   │   ├── services/      # Бизнес-логика
│   │   ├── utils/         # Утилиты
│   │   └── index.ts       # Точка входа

│

│   └── STRUCTURE.md
│
├── package.json           # Root package.json (workspaces)
└── README.md
```

## Frontend структура

### Компоненты
- `Layout/` - Основная разметка страницы
- `Header/` - Шапка сайта с навигацией
- `Footer/` - Подвал сайта
- `Button/` - Переиспользуемые кнопки
- `Card/` - Карточки для отображения информации
- `Modal/` - Модальные окна

### Страницы
- `Home/` - Главная страница
- `Services/` - Страница услуг
- `Doctors/` - Страница врачей
- `Appointment/` - Запись на прием
- `Contacts/` - Контакты

### Сервисы
- `api/` - API клиенты для взаимодействия с backend

## Backend структура

### Routes
- `/api/health` - Проверка работоспособности
- `/api/doctors` - Управление врачами
- `/api/services` - Управление услугами
- `/api/appointments` - Управление записями
- `/api/contacts` - Обработка контактов

### Controllers
- Обработка HTTP запросов
- Валидация данных
- Вызов сервисов

### Services
- Бизнес-логика
- Работа с базой данных
- Внешние API интеграции

## Планируемые функции

1. **Управление врачами**
   - Список врачей
   - Профили врачей
   - Расписание работы

2. **Услуги**
   - Каталог услуг
   - Описания и цены
   - Онлайн запись

3. **Запись на прием**
   - Выбор врача и услуги
   - Выбор даты и времени
   - Подтверждение записи

4. **Контакты**
   - Форма обратной связи
   - Контактная информация
   - Карта расположения

5. **Административная панель** (опционально)
   - Управление записями
   - Управление врачами и услугами
   - Статистика
